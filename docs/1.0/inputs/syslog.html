syslog

Read syslog messages as events over the network. This input is a good choice if you already use syslog today. It is also a good choice if you want to receive logs from appliances and network devices where you cannot run your own log collector.

Synopsis

This is what it might look like in your config file: 

inputs {
  syslog {
    host => ... # string, default: "0.0.0.0"
    port => ... # number, default: 514
  }
}

Details

host

  • Value type is string
  • Default value is "0.0.0.0"
The address to listen on

port

  • Value type is number
  • Default value is 514
The port to listen on
