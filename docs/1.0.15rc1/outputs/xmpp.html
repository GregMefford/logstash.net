xmpp

This output allows you to pull metrics from your logs and ship them to XMPP/Jabber.

Synopsis

This is what it might look like in your config file: 

output {
  xmpp {
    host => ... # string
    message => ... # string (required)
    password => ... # password (required)
    resource => ... # string (required)
    targets => ... # array (required)
  }
}

Details

host

  • Value type is string
  • There is no default value for this setting.

The xmpp server to connect to. This is optional. If you omit this setting, the host on the JID is used. (foo.com for user@foo.com)

message (required setting)

  • Value type is string
  • There is no default value for this setting.

The message to send. This supports dynamic strings like %{@source_host}

password (required setting)

  • Value type is password
  • There is no default value for this setting.

The xmpp password for the JID.

resource (required setting)

  • Value type is string
  • There is no default value for this setting.

Connection information for server

targets (required setting)

  • Value type is array
  • There is no default value for this setting.

The targets to send messages to (users, chat rooms, etc)
