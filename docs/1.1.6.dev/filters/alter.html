alter

Status: experimental

The alter filter allows you to do general alterations to fields that are not included in the normal mutate filter.

NOTE: The functionality provided by this plugin is likely to be merged into the 'mutate' filter in future versions.

Synopsis

This is what it might look like in your config file: 
filter {
  alter {
    add_field => ... # hash (optional), default: {}
    add_tag => ... # array (optional), default: []
    coalesce => ... # array (optional)
    condrewrite => ... # array (optional)
    condrewriteother => ... # array (optional)
    exclude_tags => ... # array (optional), default: []
    remove_tag => ... # array (optional), default: []
    tags => ... # array (optional), default: []
    type => ... # string (optional), default: ""
  }
}

Details

add_field

  • Value type is hash
  • Default value is {}

add_tag

  • Value type is array
  • Default value is []

coalesce

  • Value type is array
  • There is no default value for this setting.

condrewrite

  • Value type is array
  • There is no default value for this setting.

condrewriteother

  • Value type is array
  • There is no default value for this setting.

exclude_tags

  • Value type is array
  • Default value is []

remove_tag

  • Value type is array
  • Default value is []

tags

  • Value type is array
  • Default value is []

type

  • Value type is string
  • Default value is ""
