anonymize

Status: experimental

Anonymize fields using by replacing values with a consistent hash.

Synopsis

This is what it might look like in your config file: 
filter {
  anonymize {
    add_field => ... # hash (optional), default: {}
    add_tag => ... # array (optional), default: []
    algorithm => ... # ["SHA1", "SHA224", "SHA256", "SHA384", "SHA512", "MD4", "MD5", "MURMUR3", "IPV4_NETWORK"] (required), default: "SHA1"
    exclude_tags => ... # array (optional), default: []
    fields => ... # array (required)
    key => ... # string (required)
    remove_tag => ... # array (optional), default: []
    tags => ... # array (optional), default: []
    type => ... # string (optional), default: ""
  }
}

Details

add_field

  • Value type is hash
  • Default value is {}

add_tag

  • Value type is array
  • Default value is []

algorithm (required setting)

  • Value can be any of: "SHA1", "SHA224", "SHA256", "SHA384", "SHA512", "MD4", "MD5", "MURMUR3", "IPV4_NETWORK"
  • Default value is "SHA1"

exclude_tags

  • Value type is array
  • Default value is []

fields (required setting)

  • Value type is array
  • There is no default value for this setting.

key (required setting)

  • Value type is string
  • There is no default value for this setting.

remove_tag

  • Value type is array
  • Default value is []

tags

  • Value type is array
  • Default value is []

type

  • Value type is string
  • Default value is ""
