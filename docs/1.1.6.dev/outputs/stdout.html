stdout

Status: stable

Synopsis

This is what it might look like in your config file: 
output {
  stdout {
    debug => ... # boolean (optional)
    debug_format => ... # ["ruby", "json", "dots"] (optional), default: "ruby"
    exclude_tags => ... # array (optional), default: []
    fields => ... # array (optional), default: []
    message => ... # string (optional), default: "%{@timestamp} %{@source}: %{@message}"
    tags => ... # array (optional), default: []
    type => ... # string (optional), default: ""
  }
}

Details

debug

  • Value type is boolean
  • There is no default value for this setting.

debug_format

  • Value can be any of: "ruby", "json", "dots"
  • Default value is "ruby"

exclude_tags

  • Value type is array
  • Default value is []

fields

  • Value type is array
  • Default value is []

message

  • Value type is string
  • Default value is "%{@timestamp} %{@source}: %{@message}"

tags

  • Value type is array
  • Default value is []

type

  • Value type is string
  • Default value is ""
