riak

Status: experimental

Riak is a distributed k/v store from Basho. It's based on the Dynamo model.

Synopsis

This is what it might look like in your config file: 
output {
  riak {
    bucket => ... # array (optional), default: ["logstash-%{+YYYY.MM.dd}"]
    bucket_props => ... # hash (optional)
    enable_search => ... # boolean (optional), default: false
    enable_ssl => ... # boolean (optional), default: false
    exclude_tags => ... # array (optional), default: []
    fields => ... # array (optional), default: []
    indices => ... # array (optional)
    key_name => ... # string (optional)
    nodes => ... # hash (optional), default: {"localhost"=>"8098"}
    proto => ... # ["http", "pb"] (optional), default: "http"
    ssl_opts => ... # hash (optional)
    tags => ... # array (optional), default: []
    type => ... # string (optional), default: ""
  }
}

Details

bucket

  • Value type is array
  • Default value is ["logstash-%{+YYYY.MM.dd}"]

bucket_props

  • Value type is hash
  • There is no default value for this setting.

enable_search

  • Value type is boolean
  • Default value is false

enable_ssl

  • Value type is boolean
  • Default value is false

exclude_tags

  • Value type is array
  • Default value is []

fields

  • Value type is array
  • Default value is []

indices

  • Value type is array
  • There is no default value for this setting.

key_name

  • Value type is string
  • There is no default value for this setting.

nodes

  • Value type is hash
  • Default value is {"localhost"=>"8098"}

proto

  • Value can be any of: "http", "pb"
  • Default value is "http"

ssl_opts

  • Value type is hash
  • There is no default value for this setting.

tags

  • Value type is array
  • Default value is []

type

  • Value type is string
  • Default value is ""
